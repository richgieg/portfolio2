  • Neighborhood Map

    Knockout Design Patterns jQuery AJAX Google Maps API

    A required project for earning Udacity's Front-End Web Developer Nanodegree

    Open GitHub

    Before setting out to complete this project I took two Udacity JavaScript courses: "Intro to AJAX" and "JavaScript Design Patterns." Throughout these courses I learned how to fetch data asynchonously from an API server as well as how to implement design patterns to ensure the maintainability of front-end code. The Neighborhood Map project gave me a chance to apply my new knowledge and skills in order to create a single-page web application featuring a Google Map containing markers for New York City subway stations. The app features a list of the stations that can be filtered, and as they are filtered, the map's markers are updated. When a map's marker is clicked, an info window pops up, which displays a list of routes that are associated with the station, as well as images from Flickr that are geotagged within 1 km of the station. After completing the two previously-mentioned courses and this project, I can confidently perform the following:

    • Retrieve data from an API asynchronously with AJAX
    • Handle errors that can occur with AJAX calls
    • Separate concerns by following an MVC, MVP, or MVVM pattern
    • Employ the Knockout.js framework for automatic DOM manipulation
    • Utilize object-oriented design principles and patterns
    • Render an app responsively to handle various screen sizes

    By completing this project I have demonstrated my ability to utilize AJAX to interact with APIs, follow design patterns to create logical and maintainable front-end code, as well as employ the Knockout.js framework to create rich user experiences.

  • Item Catalog

    Python Flask SQLAlchemy SQL Database Design OAuth

    A required project for earning Udacity's Full-Stack Web Developer Nanodegree

    Open GitHub

    Prior to starting this project I completed two Udacity full-stack web developer courses: "Full Stack Foundations" and "Authentication and Authorization with OAuth." I thoroughly enjoyed learning how to use the Flask framework as well as SQLAlchemy. Throughout these courses I learned how to do the following:

    • Create a web application using Flask
    • Harness the power of Flask's features
    • Create and interact with databases using SQLAlchemy
    • Utilize Google's OAuth2 API to implement an authentication system
    • Implement an authorization system
    • Build a RESTful web application
    • Implement CRUD (create, read, update, delete) methods
    • Build forms for user input
    • Perform client side and server side input validation
